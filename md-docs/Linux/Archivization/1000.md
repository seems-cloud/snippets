#### zstd

```bash
zstd -z -T32 -f -M2500MB file.txt
```

#### lz4

```bash
lz4 -f -T32 file.txt file.txt.lz4
```

#### gzip
```bash
tar -czf socradar.txt.tgz socradar.txt
tar cf - file.txt | pv | gzip -1 > file.txt.tgz
```

#### pigz
```bash
tar cf - socradar.txt | pv | pigz -p16 -k --fast > socradar.txt.tar.gz

```
