#### zstd

```bash
zstd -z -T32 -f -M2500MB file.txt
```

#### lz4

```bash
lz4 -f -T32 file.txt file.txt.lz4
```

#### gzip
```bash
tar -czf file.txt.tgz file.txt
tar cf - file.txt | pv | gzip -1 > file.txt.tgz
```

#### pigz
```bash
pigz -p16 -k --fast
tar cf - file.txt | pv | pigz -p16 -k --fast > file.txt.tar.gz
```
